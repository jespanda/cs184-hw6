Original Room

HW 4 Specifications (What we based HW6 on)

Enough objects for the scene to be considered complete

  • The basic setup for our scene will be one of those "floating boat" type sushi bars
  • T We'll have a number of different kinds of sushi on top of (textured) rice clumps
  • TBasic setting for a restaurant dining table

One object created by hand

  • Bar, with "river" for boats
  • Stools
  • Table Lamps
  • Others as needed

One object loaded from a geometric format such as an .off file or a maya .obj

  • Maneki Neko cat on counter

All placement and scaling "by hand" (or with a code loop)

  • Using scene.txt format, modified and enhanced

At least one object must be textured with a suitable image

  • wood for floor
  • water for river
  • blue tiling for roof

At least two objects with two different textures. Textures must be placed to fit well on the objects

  • paintings hanging on the wall
  • wooden floor and doors

At least one object must be shiny, one must be dull

  • Shiny: Table tops, stool tops
  • Dull: Wood for the black counter.

At least one directional light

  • Global "ceiling" light

At least one point light source

  • Table lamps
  • Roof point lights

At least one object to be instantiated more than once.

  • Stools
  • Sushi plates

At least half the objects in the scene (not counting duplicates) must have correct normals so they interact with the lights correctly.

At least one shading/lighting effect

  • Changing textures for paintings

Some mechanism to turn off all textures.

  • e toggles textures.

User Interaction:

  • Use both the mouse and keyboard to allow the viewer to move about the scene.
  • Look around their current position (pivot around the eye point).
  • Move forward and backwards
  • Other appropriate translations if you wish.
  • Zoom, Translate, Use mouse
  • Currently have "always viewing center" with wheel based scaling
  • Will shift to have WASD/UpDownLeftRight controls as in a first person game with mouse helping to control center point
